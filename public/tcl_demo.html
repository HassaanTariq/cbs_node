SQL TCL Examples

Run these SQL commands in MySQL to see TCL in action:

-- COMMIT Example
START TRANSACTION;
UPDATE Account SET balance = balance + 1000 WHERE accountno = 1;
COMMIT;

-- ROLLBACK Example
START TRANSACTION;
UPDATE Account SET balance = balance - 50000 WHERE accountno = 1;
ROLLBACK;

-- SAVEPOINT Example
START TRANSACTION;
UPDATE Account SET balance = balance - 100 WHERE accountno = 1;
SAVEPOINT sp1;
UPDATE Account SET balance = balance - 200 WHERE accountno = 1;
ROLLBACK TO SAVEPOINT sp1;
COMMIT;

TCL Concepts
COMMIT

Makes all changes in the current transaction permanent

  • Used after successful operations
  • Cannot be undone
  • Example: Successful deposit
ROLLBACK

Undoes all changes in the current transaction

  • Used when errors occur
  • Restores database consistency
  • Example: Insufficient funds
SAVEPOINT

Allows partial rollback within a transaction

  • Creates restore points
  • Enables complex transaction logic
  • Example: Multi-step operations
