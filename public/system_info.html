Database Schema
Table Description Key Fields

Customer Bank customer information customerid, fullname, email

Account Bank accounts with balances accountno, customerid, type, balance
TransactionLog All financial transactions transactionid, accountno, type, amount
AuditLog System activity and changes logid, userid, action, description
API Endpoints
Method Endpoint Description
POST /api/customers Create new customer

GET /api/accounts List all accounts

Transaction Control (TCL) Features
COMMIT

Automatic commit after successful transactions ensures data persistence.

ROLLBACK

Automatic rollback on errors maintains database consistency.

SAVEPOINT

Partial rollback capability for complex transaction scenarios.
